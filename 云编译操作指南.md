# Simple Live APK 云编译指南

> 使用 GitHub Actions 免费云端编译，无需在本地安装任何开发环境

---

## 📦 准备工作

你需要一个 GitHub 账号（免费注册：https://github.com）

---

## 🚀 步骤一：创建 GitHub 仓库并上传代码

### 方法一：使用 GitHub 网页上传（推荐，无需安装Git）

1. 登录 GitHub，点击右上角 **「+」→「New repository」**
2. 填写仓库名，例如 `simple-live-build`
3. 选择 **Public（公开）**，点击 **「Create repository」**
4. 在新建的仓库页面，点击 **「uploading an existing file」**
5. 将以下整个文件夹打包成zip后拖入上传区域：
   ```
   dart_simple_live-dev_v1.11.4/  （就是这个解压后的整个文件夹）
   ```
   ⚠️ 注意：GitHub网页上传有100MB文件大小限制，建议用方法二

### 方法二：安装 Git 后命令行上传（推荐大型项目）

1. 下载安装 Git：https://git-scm.com/download/win
2. 打开 PowerShell，执行以下命令：

```powershell
# 进入项目目录
cd "C:\Users\86151\WorkBuddy\20260508092846\simple_live\dart_simple_live-dev_v1.11.4"

# 初始化 git
git init
git add .
git commit -m "first commit"

# 关联你的 GitHub 仓库（把 YOUR_USERNAME 换成你的用户名）
git remote add origin https://github.com/YOUR_USERNAME/simple-live-build.git
git branch -M main
git push -u origin main
```

---

## 🚀 步骤二：触发编译

上传代码后，GitHub Actions 会**自动开始编译**。

你也可以手动触发：
1. 进入你的仓库页面
2. 点击顶部 **「Actions」** 标签
3. 左侧找到 **「Build Android APK」**
4. 点击 **「Run workflow」→「Run workflow」**

---

## 🚀 步骤三：等待编译完成并下载 APK

1. 在 Actions 页面查看编译进度（大约需要 **5~15 分钟**）
2. 编译成功后，点击对应的工作流运行记录
3. 页面底部 **「Artifacts」** 区域找到 **「SimpleLive-APK-v1.11.4」**
4. 点击下载，解压后得到 APK 文件

### APK 文件说明

| 文件名 | 说明 | 推荐 |
|--------|------|------|
| `app-arm64-v8a-release.apk` | 64位ARM（2018年后大部分手机）| ✅ 优先安装这个 |
| `app-armeabi-v7a-release.apk` | 32位ARM（老手机）| 老手机用这个 |
| `app-release.apk` | 通用版（所有架构）| 文件较大，兼容性最好 |

---

## 📱 安装到手机

1. 将 APK 文件传到手机（微信/QQ传文件，或USB连接）
2. 手机端点击 APK 文件安装
3. 如提示"未知来源"，需要在手机设置中允许安装未知来源应用

---

## ❓ 常见问题

**Q：编译失败怎么办？**
A：点击失败的工作流，查看红色步骤的错误日志，把错误信息发给我看看。

**Q：GitHub Actions 免费吗？**
A：公开仓库完全免费，私有仓库每月有2000分钟免费额度（编译一次约10分钟，够用）。

**Q：APK 保存多久？**
A：工作流中设置了7天，7天内请下载保存。
